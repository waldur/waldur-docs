=== "HTTPie"

    ```bash
    http \
      POST \
      https://api.example.com/api/marketplace-demo-presets/load/my-awesome-resource/ \
      Authorization:"Token YOUR_API_TOKEN"
    ```

        base_url="https://api.example.com", token="YOUR_API_TOKEN"
